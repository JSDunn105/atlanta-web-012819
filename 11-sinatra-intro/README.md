### Routing

- `get '/url'`
- `post '/url'`
- `get '/url/:match'`
- `post '/url/:match'`
- match top to bottom

### ERB

- Template language
- ruby _inside_ html
- `<% %>` runs ruby
- `<%= %>` runs ruby _and shows the result_

## REST

- "Representational State Transfer"
- CRUD over HTTP
- routes match actions

Actions:
- index
- show
- new
- create
- edit
- update
- destroy


## MVC
- "Model, View, Controller"
- Separate responsibilities
- Model - it's for data
- View - it's for the view
- Controller - it's for actions
